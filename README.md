# meta-tags-generator

To install dependencies:

```bash
bun install
```

To register:

```bash
bun link
bun link meta-tags-generator
```

To run:

```bash
bunx meta-generate
```

This project was created using `bun init` in bun v1.0.16. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
